# Angular Practice Lab

Welcome to your practice lab! You have the entire `tutorials` folder from the Angular repository copied here for reference.

## How to Practice

1.  **Pick a Tutorial**:
    -   `tutorials/learn-angular`: Short, focused steps on Angular fundamentals.
    -   `tutorials/first-app`: Build a complete "Homes" application.

2.  **Read the Instructions**:
    Each step has a `README.md` or `description.md` (check the subfolders in `tutorials/`).

3.  **Apply the Code**:
    Open the `src` folder of a step (e.g., `tutorials/learn-angular/steps/1-components-in-angular/src/app/app.ts`).
    Copy the code and paste it into your `src/app/app.ts` file.

4.  **Run the Project**:
    In your terminal, navigate to this folder and run:
    ```bash
    npm install
    npm start
    ```
    Then open `http://localhost:4200`.

## Why this works
We've set up a minimalist Angular project structure that matches the style used in the official tutorials. This allows you to focus on the code without worrying about complex build configurations.

Happy Coding!
